# Домашнє завдання №1. Основи Linux і робота в терміналі

### Завдання 1. Базові команди

1. Покажи вміст домашнього каталогу:
   `ls ~`

2. Перейди у каталог Downloads і покажи його вміст:
   `cd ~/Downloads`
   `ls`

3. Створи порожній файл (без touch):
   `> empty.txt`

4. Переглянь вміст файлу:
   `cat empty.txt`

5. Перейди у домашній каталог абсолютним шляхом:
   `cd /home/username`

6. Перейди у домашній каталог відносним шляхом:
   _(перебуваючи в каталозі Downloads)_
   `cd ..`

### Завдання 2. Робота з документацією

Виконані команди:
`man ls`
`help cd`
`man cat`
`man man`
`cp --help`
`mv --help`

Відповіді на питання:

1. **Який ключ ls показує приховані файли?**
   Для цього використовується ключ `-a` або `--all`. У Linux приховані файли та каталоги починаються з крапки (наприклад, `.bashrc` або `.git`), і стандартна команда `ls` їх ігнорує. Ключ `-a` змушує систему вивести абсолютно весь вміст директорії.

2. **Який ключ у cat нумерує рядки?**
   За нумерацію рядків відповідає ключ `-n` (або `--number`). Наприклад, команда `cat -n filename.txt` виведе вміст файлу, де перед кожним рядком буде стояти його порядковий номер. Це дуже зручно під час читання логів або коду.

3. **Чим відрізняються man і --help?**

- `man` (manual) — це системна утиліта (пейджер), яка відкриває повноцінний, детальний посібник користувача для вказаної команди в окремому вікні перегляду. Там можна шукати текст, гортати сторінки клавішами `Space`/`b`, а для виходу потрібно натиснути клавішу `q`.
- `--help` — це аргумент команди, який викликає вбудовану коротку довідку. Вона не відкриває нових вікон, а миттєво виводить скорочений список основних ключів та синтаксис прямо у стандартний вивід термінала, дозволяючи одразу продовжити вводити наступні команди.

### Завдання 3. Міні-сценарій

Сценарій із 6 команд:

1. Перейти в каталог:
   `cd ~/Documents`

2. Створити там один файл:
   `> my_script.txt`

3. Переглянути список файлів:
   `ls`

4. Перейти в інше місце:
   `cd /tmp`

5. Повернутись назад:
   `cd -`

6. Переглянути документацію однієї з команд:
   `man ls`
